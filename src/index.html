Test Card Patterns

Last 4 digits determine behavior:

  • 1st: Version (0=all, 1=v2.1, 2=v2.2, 3=v2.3)
  • 2nd: 3DS Method (0=yes, 1=no, 2=timeout)
  • 3rd: Outcome (0-6=frictionless, 7=challenge)
  • 4th: Challenge type (0=auto-pass, 1=auto-fail, 2=manual)
